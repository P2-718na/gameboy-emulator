Constructor & Destructor Documentation

◆ Gameboy()

gb::Gameboy::Gameboy ( const Binary &  rom)
explicit

Instantiate the emulator with given ROM inserted. Emulation will start at $0000 and go through the boot rom; then, it will load the game (exactly as the original hardware did).

Parameters
romBinary ROM of a gameboy game. Has to be one of MBC0, MBC1 or MBC3 types. It has to be a valid gameboy ROM (only some basic checks are performed).
Exceptions
ROMbinary is either invalid data or of a unsupported MBC type.

Parameters

◆ skipBoot()

void gb::Gameboy::skipBoot ( )

Skips execution to the end of boot ROM and disables it. Tries to initialize all registers and address bus addresses to their correct values. The documentation states that these values should not be relied upon by ROMs and that they may not be universal across devices.
